# Campus Notification System

## Description
A backend-based notification management system developed using Node.js and Express.js. The system fetches notifications from an external API and prioritizes them based on category.

## Features
- Fetch campus notifications
- Priority-based sorting
- REST API integration
- Logging middleware
- Modular backend architecture

## Technologies Used
- Node.js
- Express.js
- Axios

## API Endpoint

GET /priority-notifications
